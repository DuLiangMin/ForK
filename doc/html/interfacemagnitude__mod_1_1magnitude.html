Subroutine for computing the optimal magnitude associated with a function-only Kriging model. It is used when the length scale and noise ratio are determined and the magnitude is needed. This code is repeated from the likelihood_mle subroutine. If the magnitude formula changes, both this and likelihood_mle need to be modified.

The optimal magnitude given lenght scales and noise ratio is given as:

\[\sigma^{2} = \frac{(Y-H^{T} \beta)^{T} \hat{K}^{-1} (Y-H^{T} \beta )}{N} \]

where \(\hat{K}\) is the unit magnitude covariance matrix, the elements of which are given as:

\[ \hat{K}_{i,j} = k(\vec{x}_{i}, \vec{x}_{j},\theta) + \Gamma \delta_{i,j}\]

This subroutine is NOT used within the likelihood_mle subroutine and is used only when the length scale is known and the magnitude is needed.

Parameters:
in) ndim : The dimension of the problem
in) ntot : The number of training points
in) X : Location of the training points (size=[ndimxntot])
in)Y : Funcation values for the training points (size=[ntot])
in)stot : Number of Terms in the regression
in)H: The collocation matrix for the regression (size=[stotxntot])
in) theta : Length scale in each direction (size=[ndim])
in) gamma : Ratio of Noise Magnitude to covariance matrix (added to diagonal of covariance matrix with magnitude of 1)
out) sigma : Optimal covariance magnitude ( \(\sigma\))
