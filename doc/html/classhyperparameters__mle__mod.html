subroutine hyperparameters_mle_func (ndim, ntot, X, Y, stot, H, theta, sigma, sigmaN, logpy)
 Subroutine used to set up the optimization required to determine hyperparameters for a function-only Kriging model. Optimization is performed only for the length scales used in the covariance function and covariance magnitude is determined explicitly. Optimization bounds for the length scales are found here. The hard-coded noise level used to keep the covariance matrix positive-definite is also specified here.
subroutine hyperparameters_mle_grad (ndim, ntot, X, Y, gtot, pts, dims, dY, stot, H, theta, sigma, sigmaN, sigmaNG, logpy)
 Subroutine used to set up the optimization required to determine hyperparameters for a gradient-enhanced Kriging model. Optimization is performed only for the length scales used in the covariance function and covariance magnitude is determined explicitly. Optimization bounds for the length scales are found here. The hard-coded noise level for the function and gradient used to keep the covariance matrix positive-definite are also specified here.
