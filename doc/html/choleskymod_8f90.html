Detailed Description

Module containing matrix manipulation subroutines such as Cholesky factorization and matrix multiply. For the most part, the functions contained here are mere wrappers for LAPACK calls. Commented out functions contain the "dumb" versions of these LAPACK calls and are used primarily for debugging purposes.
