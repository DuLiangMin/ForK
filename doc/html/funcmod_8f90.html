Detailed Description

This module contains the function calls that can be used in conjunction with the optimization algorithms patternsearch and simplexsearch. The call to the subroutine "func" is made by the subroutine funcwrapper. The subroutine is overloaded based on the type of the arg argument. For each arg type, a different function is called (either toy, likelihood, likelihood_mle, or kriging). This module in addition to argument and funcwrapper must be modified if optimization is to be performed on different objectives.
