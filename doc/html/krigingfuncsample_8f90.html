subroutine krigingfuncsample (mtot, Ym, S, U, Yp)
 This subroutine extracts a sample from the Kriging models output distribution. This sample extraction is given as:

\[ Y_{p} = Y_{m} + L u\]

where \(Y_{m}\) is the mean of the Kriging function's output (the result of krigingfuncpredict) and \(u\) is a vector of normally distributioned values of length \(M\) (normal distribution with zero mean and standard deviation of 1). The matrix \(L\) is the result of cholesky performed on the covariance matrix of the Kriging model's output distribution (the result of krigingfunccovar).

Detailed Description

This subroutine samples from the Kriging model's output distribution. It requires krigingfunccovar and krigingfuncpredict to be call first and cholesky must be performed on the output distribution covariance matrix.

This subroutine extracts a sample from the Kriging models output distribution. This sample extraction is given as:

\[ Y_{p} = Y_{m} + L u\]

where \(Y_{m}\) is the mean of the Kriging function's output (the result of krigingfuncpredict) and \(u\) is a vector of normally distributioned values of length \(M\) (normal distribution with zero mean and standard deviation of 1). Parameters:
in) mtot: The number of test points to be predicted
in) Ym : The mean of the output distribution (the result of krigingfuncpredict) (size=[mtot])
in) S : The cholesky factorization of the covariance matrix from the output distribution (krigingfunccovar) stored in the lower triangle. (size=[mtotxmtot])
in) U : A vector of normally distributed random numbers (size=[mtot])
out) Yp : The output sample from the Kriging model. This subroutine should be called multiple times to acquire a large number of output samples, allowing for global predictions to be calculated multiple times and a output distribution constructed.
