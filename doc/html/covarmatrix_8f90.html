Detailed Description

This module contains the subroutines used to compute the covariance matrix. For efficiency, different subroutines are used for the different covariance matrices needed in the code. Although the naming may be confusing, this module contains the subroutines required to calculate the block covariance matrix needed for a gradient-enhanced Kriging model.
